# Shopping Centre Management System

A C++ console application for managing businesses, shops, restaurants, and employees in a shopping centre.

## Features

- Add businesses and employees
- Manage restaurants and different shop types
- Display business and employee details
- Show shop statistics by category
- Search employees by name
- Find restaurants by seating capacity
- Find the largest business by size

## Requirements

- C++23 compatible compiler
- CMake 3.31 or later

## Build and Run

```bash
cmake -S . -B build
cmake --build build
```

Run:
./build/242_A2
On Windows, the executable may be located at:
build/Debug/242_A2.exe
